## 3. Fehler melden & Änderungen wünschen

Auf eurer Website befindet sich **rechts unten ein oranger Button** 🐛.

**So funktioniert es:**

1. Button anklicken
2. Kurz beschreiben was nicht stimmt oder was ihr wünscht  
   (z.B. *„Der Text bei Wert-Karte 2 hat einen Tippfehler"* oder *„Können wir noch einen Partner hinzufügen?"*)
3. Auf **„Senden"** klicken

Das landet direkt bei eurem Entwickler — mit Screenshot des aktuellen Stands, damit der Fehler sofort nachvollzogen werden kann.

**Tipp:** Ihr könnt diesen Button auch nutzen, um **Änderungswünsche** einzugeben die ihr nicht selbst im Admin machen könnt (z.B. Design-Anpassungen, neue Funktionen).

## 4. Events verwalten

Events sind der einzige Bereich mit etwas mehr Funktionen:

- **Neues Event anlegen** → Formular am Ende der Event-Liste ausfüllen
- **Event bearbeiten** → auf den Bearbeiten-Button klicken
- **Status setzen:**
  - `upcoming` = wird auf der Website als nächstes Event angezeigt
  - `past` = erscheint in der Vergangenheitsliste
- Das Event mit Status `upcoming` und dem **frühesten Datum** erscheint automatisch in der großen Instagram-Kachel oben links

## 5. Fotos hochladen (Team)

Im Bereich **👥 Team** könnt ihr Fotos von euch hochladen:

1. Auf **„Datei auswählen"** klicken
2. Foto auswählen (JPG oder PNG, max. 8 MB)
3. Auf **„Hochladen"** klicken

Mit dem **🗑 Löschen-Button** könnt ihr das aktuelle Foto entfernen.

> **Hinweis:** Die Fotos werden im Hintergrund gespeichert und sind für eine spätere Version (V2) der Website vorbereitet. Ihr könnt sie schon hochladen.

## 6. Eigene Domain (zukünftig)

Sobald ihr eine eigene Domain habt — z.B. **www.minikosmosstuttgart.de** — kann die Website einfach umgezogen werden:

- Alles bleibt gleich: Admin, Inhalte, Fotos, Events — nichts geht verloren
- Es wird nur die Adresse geändert, unter der die Website erreichbar ist
- Der Umzug dauert etwa 30 Minuten technische Arbeit + DNS-Propagation (bis 24h)

Meldet euch einfach wenn es so weit ist — das wird koordiniert.

## 7. Häufige Fragen

**Ich habe das Passwort vergessen.**  
→ Meldet euch beim Entwickler, das Passwort kann jederzeit zurückgesetzt werden.

**Ich habe etwas gespeichert und es sieht falsch aus.**  
→ Einfach nochmal in den Admin-Bereich gehen und den Text korrigieren. Es gibt keine kaputte Aktion.

**Die Website lädt nicht.**  
→ Kurz warten und neu laden. Falls es länger nicht funktioniert, über den Fehler-Button Bescheid geben oder direkt melden.

**Kann ich die Website auf dem Handy bearbeiten?**  
→ Ja, der Admin-Bereich funktioniert auch auf dem Smartphone, ist aber am Laptop/Computer komfortabler.
